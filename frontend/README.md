# OpportuneX Frontend

Campus-focused Internship & Project Opportunity Platform frontend.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will run on http://localhost:3000

## Backend Connection

Make sure the FastAPI backend is running at http://127.0.0.1:8000

## Features

- JWT Authentication (Login/Register)
- Student Dashboard with recommended opportunities
- Opportunity browsing with fit scores
- Apply to opportunities
- Filter by internal/external opportunities

## Tech Stack

- React 18
- Vite
- React Router DOM
- Plain CSS
- Fetch API
